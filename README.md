# HR_ATTRITION_DASHBOARD-Major-Project-
This project analyzes employee attrition using HR data and presents insights through an interactive Power BI dashboard to help improve employee retention.
HR Employee Attrition Analysis – Power BI Dashboard

 Project Description:

Employee attrition is a major concern for organizations as it affects operational stability, recruitment costs, and team productivity.
This project presents an interactive HR Attrition Dashboard built in Power BI to analyze employee turnover and identify the underlying factors influencing attrition.

The analysis focuses on demographics, compensation, career growth, and employee engagement, enabling HR teams to gain actionable insights.

     
 Project Goals:

Measure overall employee attrition levels

Analyze attrition across departments and job roles

Study demographic trends related to employee turnover

Evaluate the impact of salary, promotions, and experience

Understand how engagement and satisfaction affect attrition



---

 Dataset Information

Dataset Name: IBM HR Analytics Employee Attrition Dataset

Number of Records: 1,470 employees

Data Includes: Employee demographics, job details, salary structure, performance ratings, and satisfaction scores



---

 Data Modeling Approach

Model Type: Star Schema

Fact Table: Fact_Attrition (Employee-level attrition metrics)

Dimension Tables:

Employee

Department

Compensation

Experience

Satisfaction

Education


Measures Table: Dedicated table for KPIs and DAX measures


This structure improves performance, clarity, and scalability.

 Insights Gained

Higher attrition observed among lower-income and early-career employees

Employees experiencing delayed promotions tend to leave more frequently

Overtime and poor work-life balance strongly correlate with attrition

Attrition is present across all performance levels, including high performers




 Key Learnings

Effective use of data modeling improves analytical clarity

Importance of correct aggregation methods in Power BI

Binning continuous variables enhances visualization

Dashboard storytelling is as important as technical accuracy
